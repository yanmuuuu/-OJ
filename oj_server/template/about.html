OnlineJudge

C++ Distributed Online OJ Platform with Load Balanced Compile Servers

版权说明

此项目基于 C++ 分布式架构在线 OJ 平台，仅供学习使用。禁止用于商业目的。

项目简介

基于 C++17 的分布式在线编程评测系统。用户可以在线浏览题库、编写代码、提交判题并查看运行结果。系统采用前后端分离与多编译服务器负载均衡：主服务器处理 HTTP 请求与业务逻辑，编译服务器集群负责代码编译与进程隔离运行。

  • 分布式判题与负载均衡
  • 在线 Ace 代码编辑器
  • 用户注册登录与题目录入
  • MySQL 持久化题库管理
